Export Functionality Verification

To verify the export functionality is working:

  1. Run npm run dev to start the development server
  2. Open http://localhost:3000 in your browser
  3. Fill in the calendar form:
    • Select a start date
    • Choose a rotation pattern (e.g., 14/14)
    • Click "Generate"
  4. Once the calendar is generated, on desktop:
    • Look for a floating action button (FAB) in the bottom-right corner
    • Click it to open the export menu
    • Select PNG, PDF, or Add to Calendar (.ics)
    • The file should download

Expected Results:

  • ✅ FAB appears in bottom-right corner on desktop
  • ✅ Clicking FAB shows export menu with 3 options
  • ✅ PNG export creates image file
  • ✅ PDF export creates PDF document
  • ✅ Add to Calendar creates .ics file
  • ✅ Success toast notifications appear
